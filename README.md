# Epigram Generator

A full-stack application for generating, creating, and interacting with epigrams. The project demonstrates a clean separation between backend and frontend, real-time UI updates, and interactive features like liking, disliking, and auto-rotating content.

## Markov Chain Generation

The backend uses a **Markov Chain** approach to generate epigrams.

### How it works:
- A dataset of existing epigrams is used as training data.
- The generator builds a probabilistic model of word transitions.
- Each word predicts the next possible word(s) based on frequency.
- New sentences are generated by:
    1. Starting with a random word.
    2. Continuously selecting the next word based on probabilities.
    3. Stopping when a sentence-ending condition is met.

### Result:
- The generated epigrams are **statistically similar** to the dataset.
- They may not always be perfectly meaningful, but they:
    - Maintain stylistic patterns
    - Produce varied and creative outputs
    - Demonstrate probabilistic text generation techniques
